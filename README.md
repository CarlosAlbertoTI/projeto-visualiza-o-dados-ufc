# BRUMS — Tatame Universitário

Resumo curto  
Projeto de disciplina de Visualização de Dados: visualizações interativas para analisar o efeito do treino em sentimentos (pré vs pós).

## Descrição
Conjunto de visualizações (D3 v7 + Bootstrap) que explora:
- TMD (Distúrbio Total de Humor)
- Perfil "Iceberg" (radar)
- Gráfico dumbbell (mudança pré → pós)
- Gráfico slope (detalhe individual)

Interface responsiva com foco em comparação de magnitudes e leitura clara de mudanças.

## Estrutura (exemplo)
- index.html — markup principal (já fornecido)
- style.css — estilos
- script.js — lógica D3
- data/ — CSV(s) de entrada
- README.md — este arquivo

## Tecnologias
- D3.js v7
- Bootstrap 5.3
- HTML5 / CSS3 / JS
